## Overview

PythonTools provides foundational modules intended for reuse across multiple projects, including:

- Common utility functions  
- Shared constants  
- Schema‑based loading and validation helpers  
- Normalization and inheritance logic  
- General‑purpose tooling patterns  

The goal is to centralize shared logic so application‑level projects (like RunUpdates) stay clean and focused.
